# Task management system

# features
- Add new tasks
- Update existing tasks
- Delete tasks
- View all tasks
- Exit program

# how to run
1. Save code as `tasks_manager.py`
2. Run: `python tasks_manager.py`
3. Follow on-screen prompts

# menu options
1 - Add task
2 - Update task
3 - Delete task
4 - View tasks
5 - Exit

# Example
Enter tasks initially → Use menu to manage → View numbered list
